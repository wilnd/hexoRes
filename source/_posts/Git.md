>### 介绍
#### 将本地目录初始化为Git仓库
·git init· 
#### 提交修改
·git add .·
提交您在本地存储库中上上传的文件
·git commit -m '备注'·
配置要上传的仓库地址url
·git remote add origin 'https://github.com/wilnd/hexoRes.git'·
将本地存储库中的更改推送到GitHub
` git push origin master`
